asCanvas(option)

概述

给匹配的Canvas元素 创建context对象。

参数:option

{name : 'context-name'}

描述:

为ID是"test"的Canvas元素创建context对象，并将创建的context对象命名为'desk'。便于以后用选择器直接选择这个context对象，例如：$('$desk')

HTML 代码:

<canvas id="test" width="507px" height="320px"></canvas>

elves 代码:

$("#test").asCanvas({
	name : 'desk'
});
