arc()

概述

canvas方法之一。在画布（canvas）的content上画圆。

参数:x, y, r, color

x, y为圆心在画布中的坐标位置。r为圆的半径。color为圆的颜色。

描述:

查找 ID 为"myDiv"的元素。

HTML 代码:

<canvas id="test" width="507px" height="320px" style="border:1px solid #F00">

elves 代码:

$(function() {
	$("#test").asCanvas({  //选择id为test的canvas元素，然后为这个canvas元素创建content对象。
		name : 'desk'  //   将创建content对象命名为desk,便于选择器选择。如下例$("$desk")。
	}).arc(10,20,5,"#09C");  //  在content对象上画圆。
	
	$("$desk").arc(20,20,5,"#f00");  //已有名为desk的canvas元素content对象，$("$desk")就可选择到。
});
