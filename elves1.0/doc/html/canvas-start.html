start()

概述

canvas方法之一。允许清空画布内的像素，即设置flush()为可用

描述:

通过start()方法的设置，使flush()方法可用

HTML 代码:

<canvas id="test" width="507px" height="320px" style="border:1px solid #F00">

elves 代码:

$(function() {
	//创建一个名为'desk'的canvas  content对象
	$("#test").asCanvas({
		name : 'desk'
	});
	
	//在这个对象上画个的圆形，圆心坐标（20px,20px）,半径5px,颜色红色
	$("$desk").arc(20,20,5,"#f00"); 
	
    $("$desk").stop();  //通过stop()方法的设置，使flush()方法禁用
	
    //一秒钟后，清除画布内的像素
	setTimeout(function(){
		$("$desk").start();  //通过start()方法的设置，使flush()方法可用
		$("$desk").flush();  //清空"$desk"内的像素。
	},1000)
});
