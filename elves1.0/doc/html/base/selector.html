Dom Selector 概述

根据给定的选择器，选择对象。selector分为三种类型。

描述:

支持css3选择器,根据css3选择器选择Dom

HTML 代码:

<div id="notMe"><p>id="notMe"</p></div>
<div id="myDiv">id="myDiv"</div>

elves 代码:

$("#myDiv");

结果:

[ <div id="myDiv">id="myDiv"</div> ]

Canvas Selector 概述

根据canvas id查找Canvas对象。使用$开头后面跟上ID，例如 $id

描述:

查找canvas

HTML 代码:

<canvas id="myCanvas" width="507px" height="320px" style='border:1px solid red;'></canvas>

elves 代码:

$("#myCanvas").canvas('canvas_id');//初始化canvas
$("$canvas_id");//选择canvas

结果:

[ Canvas Object ]

Entity Selector 概述

根据entity id查找Entity对象。需要先选择canvas，再接上eneity selector，entity selector以@开头。 例如 $canvas_id@entity_id

描述:

查找entity

HTML 代码:

<canvas id="myCanvas" width="507px" height="320px" style='border:1px solid red;'></canvas>

elves 代码:

$("#myCanvas").canvas('canvas_id').entity('entity_id');//初始化canvas，并且创建entity
$("$canvas_id@entity_id");//选择entity

结果:

[ Entity Object ]
